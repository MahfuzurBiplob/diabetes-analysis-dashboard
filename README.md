# Diabetes Analysis Dashboard

## 📊 Project Overview

This project analyzes diabetes patient data using Power BI to identify key health patterns and risk factors.

---

## 📁 Dataset

* Source: Cleaned diabetes dataset
* Features include:

  * Age
  * BMI
  * Glucose level
  * Outcome (Diabetic / Non-diabetic)

---

## ⚙️ Tools Used

* Power BI
* CSV dataset

---

## 📈 Dashboard Features

* Age group analysis
* BMI category analysis
* Glucose distribution
* Gender filter

---

## 🔍 Key Insights

* Most diabetic patients are in the 40–50 age group
* Higher BMI is strongly linked with diabetes
* High glucose levels indicate higher diabetes risk
