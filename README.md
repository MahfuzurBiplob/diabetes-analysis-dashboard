# Diabetes Analysis Dashboard

## 📊 Project Overview
This project analyzes diabetes patient data using Power BI.

## 📁 Dataset
- Source: Public dataset (Kaggle)
- Records: ~2000+

## 📈 Key Metrics
- Total Patients
- Diabetic Count
- Diabetic %
- Avg Glucose
- Avg BMI

## 📊 Dashboard Features
- Age group analysis
- BMI category analysis
- Glucose distribution
- Gender filter

## 🛠 Tools Used
- Power BI
- CSV
